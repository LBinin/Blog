# Web 语义化

语义化起码是让能让机器懂。

机器懂的两个大目标么，一个是「搜索」，一个是「屏幕阅读器」。

搜索，就是 SEO；

阅读器，其实是配合浏览器的 **Web Accessibility Initiative** 具体来工作。
